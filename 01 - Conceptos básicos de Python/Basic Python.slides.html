Bucles e instrucciones condicionales¶

Podemos hacer bucles (loops) para hacer clculos y print para mostrar el resultado final. En python, el espacio es muy importante. Para delimitar el contenido de un bucle, no se utilizan llaves, ni parntesis, ni palabras clave tipo begin o end. El espacio deja claro dnde comienza y donde termina un bucle.

In [5]:
result=0
for i in range(1,10):
    print(i, end=' ')
    result+=i
print("\nLA SUMA ES: {:7d}".format(result))

result=0
for i in range(1,10):
    if (i%2==0):
        print(i, end=' ')
        result+=i
print("\nRESULT IS: {:7d}".format(result))

2 4 6 8

Subrutinas y funciones¶

Como en cualquier otro lenguaje de programacin, podemos declarar funciones y utilizarlas ms adelante en nuestro cdigo. Esto nos permite crear una librera de funciones especializadas que podemos utilizar posteriormente. Recordar que:

  • Las funciones se definen con la palabra clave def.
  • Los argumentos de la funcin se especifican entre parntesis.
  • Igual que con los bucles, el contenido de la funcin se delimita mediante el espacio.
In [6]:
def fact(x):
    if not isinstance(x, (int)) or (x<0):
        return 0
    if x==0:
        return 1
    else:
        return x*fact(x-1)
    
fact(5)

Diccionarios¶

Los diccionarios son tiles para almacenar informacin en trminos de una clave (palabra o secuencia de letras).

Los diccionarios se especifican como una secuencia separada por comas de etiquetas sus correspondientes valores, separados estos por dos puntos. La secuencia completa se delimita mediante llaves.

Por ejemplo, podramos almacenar las masas atmicas de molcular sencillas:

In [11]:
MW={'H2O': 18.02, 'CO2': 44.01, 'C2H6': 30.07}

Podemos acceder a cualquiera de los elementos del diccionario fcilmente utilizando su clave:

In [12]:
MW['C2H6']
Out[12]:
30.07

Operaciones avanzadas con diccionarios:¶

  • Se pueden recorrer mediante un LOOP
  • Se pueden ordenar por clave
  • Se pueden ordenar por valor

Resolviendo ecuaciones diferenciales con Scipy¶

El matlab de python (junto con numpy).

Una ecuación diferencial ordinaria¶

Nos proponemos resolver la ecuacin diferencial ordinaria siguiente: $$ \frac{dy}{dt}=- 0.1 y + sin(y)$$ El mtodo bsico require definir:

  • modelo: La funcin que devuelve el valor de la derivada en trminos de los valores actuales de y
  • y0: La condicin inicial
  • t: Los instantes de tiempo en los que queremos determinar la solucin.
In [23]:
%matplotlib inline
from scipy.integrate import odeint
def modelo(y, t):
    return -0.1*y+np.cos(y)

y0 = 5
t = np.linspace(0,10,100)
y = odeint(modelo, y0, t)
plt.plot(t,y)
plt.xlabel("t")
plt.ylabel("y")
Out[23]:
Text(0, 0.5, 'y')

Matemáticas simbólicas con Sympy¶

Sympy es una librera de Python para hacer clculos con matemtica simblica (de forma parecida a Maple o Mathematica). En esta librera, antes de poder operar con smbolos algebraicos, tenemos que definirlos. Los smbolos se pueden definir uno a uno o como listas. Para simplificar el uso, es importante asignar los simbolos a una variable que se llame igual que el smbolo en cuestin:

In [27]:
from sympy import *
x=symbols('x')
v, y, z = symbols('v y z')
sin(x+1)*z**y/v
